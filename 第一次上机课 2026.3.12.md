# 实验一 PostgreSQL 基本操作

## 一、实验环境准备

1. 启动电脑，选择 **win10-1 分区**
2. 在菜单栏找到 PostgreSQL，启动后进入 **pgAdmin 控制面板**

## 二、实验要求

1. 熟悉 PostgreSQL 的基本操作
2. 使用**图形化界面**创建数据库、数据表、插入数据
3. 使用 **SQL 命令**完成创建数据库、数据表、插入数据
4. 数据库**备份和恢复**

## 三、实验内容和步骤

### （1）图形化界面操作（pgAdmin）

### （2）命令行操作（SQL Shell / psql）

打开方式：开始菜单 → PostgreSQL → **SQL Shell (psql)**

启动时各项提示**直接回车跳过**，只在"口令"处输入 `admin`

#### 常用命令速查

| 功能 | 命令 |
| --- | --- |
| 创建数据库 | `CREATE DATABASE dbname;` |
| 查看所有数据库 | `\l`（小写 L） |
| 删除数据库 | `DROP DATABASE [IF EXISTS] name;` |
| 切换/进入数据库 | `\c 数据库名` |
| 查询表数据 | `SELECT * FROM 表名;` |

#### 切换到 ehall 数据库

```
postgres=# \c ehall
-- 输出：您现在已经连接到数据库 "ehall"，用户 "postgres"。
ehall=#
```

#### 创建数据表（命令行方式）

本次需要创建：**门禁（access）**、**消费（consume）**

**操作步骤**：
1. 打开 **SQL Shell (psql)** → 回车跳过提示 → 在"口令"处输入 `admin`
2. 进入 ehall 数据库：`\c ehall`
3. 复制下面的 SQL 代码 → 粘贴到命令行 → 最后打 `;` 再回车执行

> [!example]- 示例：创建 access 表
> ```sql
> CREATE TABLE ACCESS(
>     ACCESSID    INT         PRIMARY KEY  NOT NULL,
>     SID         CHAR(11)                 NOT NULL,
>     ACCESSTIME  TIMESTAMP                NOT NULL,
>     GATENO      INT
> );
> ```

> [!example]- 示例：创建 consume 表
> ```sql
> CREATE TABLE CONSUME(
>     CONSUMEID   INT         PRIMARY KEY  NOT NULL,
>     SID         CHAR(11)                 NOT NULL,
>     CONSUMETIME TIMESTAMP                NOT NULL,
>     AMOUNT      DECIMAL(8,2)             NOT NULL,
>     PLACE       VARCHAR(30)
> );
> ```

## 五、常见问题

### a) 大小写问题

| 场景 | 规则 |
| --- | --- |
| SQL **关键字**（SELECT、FROM 等） | **不区分**大小写 |
| **表名、列名**等标识符 | 默认不区分，PostgreSQL 自动转小写 |
| 标识符用**双引号**包裹时 | **区分**大小写，且必须用双引号引用 |
| **字符串值**（`WHERE name = 'John'`） | **区分**大小写 |

不区分大小写的字符串比较，使用 `ILIKE` 或 `LOWER()` 函数：

```sql
SELECT * FROM my_table WHERE LOWER(name) = LOWER('John');
SELECT * FROM my_table WHERE name ILIKE 'john';
```

### b) 命令行换行问题

SQL 语句以 `;` 结尾，**回车不会执行**，可以分多行输入，最后打 `;` 再回车才会执行 ✅

## pdf-to-obsidian-notes

一个 [Claude Code](https://claude.com/claude-code) 自定义 Skill，核心功能：

- **PDF → 笔记**：将教材、课件、论文等 PDF 转化为结构清晰的 Obsidian Markdown 笔记
- **智能补充**：基于 PDF 内容对已有笔记进行原地改写和补充，而非简单追加
- **多源融合**：同时读取课件 + 教材，教材补充内容用 callout 标记，自然融入笔记
- **格式规范**：frontmatter、callout、wikilink、LaTeX 公式等 Obsidian 语法均正确处理

详见 [pdf-to-obsidian-notes/README.md](./pdf-to-obsidian-notes/README.md)。
